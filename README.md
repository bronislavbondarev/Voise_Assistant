# Voise_Assistant
Локальный голосовой помощник на Python для ПК. Распознаеткоманды с микрофонаи выполняет действия: открывает приложения, ищет информацию в интернете и браузере благодаря библиотекам: os, time, webbrowser, keyboard, subprocess, pygame, win32api, win32con, tkinter и некоторых других. Для работы использует библиотеки распознавания(speech_recognition) и синтеза речи(edge_tts). Полностью работает офлайн, без облачных сервисов. Просто назови задачу - и он ее выполнит.
